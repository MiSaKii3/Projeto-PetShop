1- Como funciona o nosso jogo:

O nosso projeto possui três páginas: Uma com um login, outra com o jogo em si e outra com o formulário para mudar os dados.

- Na página do login, é pedido a quem vai entrar no jogo que introduza o seu username e a sua password para entrar no mesmo (com a possibilidade de mostrar ou esconder a pass);

- Na página de formulário, podemos trocar os nossos dados de acesso;

- Após a introdução dos dados corretos, quem entra é redirecionado para a página do jogo, que funciona da seguinte maneira:

O objetivo é terminar um dia de trabalho na loja (cuja duração é de 6 minutos) com ganhos positivos. Se os ganhos forem menores que 300 moedas, o jogador perde.
Durante este dia de trabalho, quem joga deve atender os diversos clientes que aparecem, fornecendo os animais ou rações que eles pretendem. 
Para além disto, também é necessário alimentar os animais de x em x tempo. (Não fizemos essa parte, kinda)
Estas ações são realizadas através do método drag and drop, que permite ao jogador arrastar os pedidos dos clientes até eles.
